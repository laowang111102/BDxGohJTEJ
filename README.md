## 项目概述

新生入学管理系统是一款集成多种功能的平台，旨在辅助高校高效地进行新生入学管理工作。系统通过前端门户、专业管理、班级管理、学生管理、商品分类与信息管理、签到管理和交流论坛等功能模块，实现全流程的管理。

## 技术栈

- 后端：Python 3.7.7，Django 2.0，pymysql，click，django-threadlocals，requests，xlrd 1.2.0，django-cors-headers
- 前端：Vue 2
- 运行环境：MySQL 5.7，Node 12，cnpm 7.1.0

## 功能模块

### 前台门户

- 提供友好的用户界面，展示入学相关信息。
- 包含模块：首页、专业介绍、入学指南、校园新闻、论坛等。

### 管理模块

#### 专业管理

- 管理专业信息，设置名称、简介、所属学院、学制等。

#### 班级管理

- 管理班级信息，创建、修改、查询和删除班级。

#### 学生管理

- 记录和管理新生信息，支持信息的批量导入和导出。

#### 商品分类管理

- 对校园销售商品进行分类，便于查找和购买。

#### 商品信息管理

- 记录和管理商品详细信息，确保信息准确性和实时性。

### 新生签到管理

- 协助新生报到签到，记录签到时间、地点等。

### 交流论坛

- 提供在线交流平台，支持发帖、评论和主题设置。

### 轮播图管理

- 管理前台门户轮播图，展示重要信息和活动宣传。
